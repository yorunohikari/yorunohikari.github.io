ヘルプ

日本の都道府県旗クイズへようこそ！

遊び方：

1. スライダーを使用してクイズに含める都道府県の数を選択します。

2. 「プレイ」ボタンをクリックしてクイズを開始します。

3. 都道府県の旗が表示され、提供されたオプションから正しい都道府県を選択する必要があります。

4. すべての質問に回答した後、結果が表示されます。

コントロール：

  • 都道府県の数を選択： スライダーを使用してクイズに含める都道府県の数を選択します。
  • プレイボタン： 選択した都道府県の数でクイズを開始します。
  • 旗の表示： 都道府県の旗が表示され、提供されたオプションから正しい都道府県名を選択する必要があります。
  • 結果の表示： すべての質問に回答した後、正解数と不正解数が表示されます。
  • もう一度プレイボタン： クイズをもう一度プレイするにはクリックします。

クイズを楽しんで、日本の都道府県に関する知識をテストしてください！
